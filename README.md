# UGC by Olla – Landingpage

Statische Landingpage für das UGC-Portfolio von Olla Gerber.
Reines HTML/CSS/JS – kein Build-Schritt nötig.

## Struktur
- `index.html` – die Seite
- `styles.css` – Styling
- `script.js` – Interaktionen (Logoband, KPI-Zähler, Portfolio-Lightbox, Mobilmenü)
- `assets/` – Bilder, Marken-Logos und Portfolio-Videos

## Lokal ansehen
`index.html` einfach im Browser öffnen (Doppelklick).

## Hosting
Geeignet für GitHub Pages, Netlify oder Cloudflare Pages.
Bei GitHub Pages: Repo-Einstellungen → Pages → Branch `main`, Ordner `/ (root)`.
Für eine eigene Domain wird eine `CNAME`-Datei im Wurzelverzeichnis ergänzt.
